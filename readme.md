# SwiftNpy
Load NumPy array files in Swift

## Suppoted format
### UInt
`UInt8`, `UInt16`, `UInt32`, `UInt64`  
They also can be read as `UInt`

### Int
`Int8`, `Int16`, `Int32`, `Int64`  
They also can be read as `Int`

### Float, Double
`Float`, `Double`
